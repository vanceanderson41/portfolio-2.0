IEEE-754 Floating Point Conversion - Assembly

I was part of a team that designed and implemented the algorithm to successfully convert base 10 floats to single precision binary representation. Any floating point decimal number representable by single precision IEEE format can be converted. The input is entered into a UART and received as ASCII and then converted to the 32-bit format defined by the standard. This effort was accomplished using PLPTool, a simulation of a RISC.
Aside from the conversion, I wrote a program that implements all IEEE-754 operations except for division and raises appropriate exception flags due to invalid operations, overflow, underflow, and inexact rounding.
